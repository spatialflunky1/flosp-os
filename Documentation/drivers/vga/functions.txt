80x25 Text Mode Functions
--------------------------
clear_output_8025()
    Clears the display to a blank black screen
kprint_8025(const char* str)
disable_cursor()
enable cursor()
set_cursor_pos(unsigned short pos)
    Position is a linear value (0: top, 80*25-1: bottom right)

    Scrolls the text onscreen "down" (moves up)
